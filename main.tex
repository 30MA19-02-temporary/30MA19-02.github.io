\title{The Development of Algebraic Model for Geometries on Manifold with Constant Intrinsic Curvature}
\shorttitle{30MA19-02}
\leftheader{Hanchai, Sakepisit}
\authorsnames{Hanchai Nonprasart, Sakepisit Maysamat}
\authorsaffiliations{Mahidol Wittayanusorn School}
\course{SCI 30196, SCI 30197, SCI 30198: Science Project}
\professor{Amornsri Amornvatcharapong, Teerapong Suksumran}
\duedate{September 15, 2022}
\abstract{%
High school geometry consists of Euclidean geometry with no intrinsic curvature
which can be represented using vectors and matrices as can be seen in linear algebra.
In other geometries such as spherical geometry which can be applied to cartography, however,
require a different model to achieve the same task.
We are therefore interested in developing a united model
which describes geometric transformations on constantly-curved manifold:
spherical geometry, Euclidean geometry, and hyperbolic geometry.
The model is defined as a class of group acting on Riemannian manifold with constant sectional curvature
where the group action is an isometry, translation, rotation, and reflection, on the manifold.
Additionally, the projection from points in the developed model to points in the existing model and,
subsequently, software to visualize the transformation on the manifold have been developed.%
}
\keywords{Lie group, isometric embedding, sectional curvature, non-Euclidean geometry, spherical geomerty, hyperbolic geometry}
